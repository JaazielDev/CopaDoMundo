## Sobre o projeto

O **Copa do Mundo** é um site desenvolvido com HTML, CSS e JavaScript puro, criado para ser uma referência visual e interativa sobre o maior evento do futebol mundial. O projeto explora a história das edições da Copa, seleções campeãs, artilheiros e muito mais — com um design limpo e totalmente responsivo.

---

## Funcionalidades

- **Histórico completo** — todas as edições da Copa do Mundo desde 1930
- **Seleções** — países participantes, títulos conquistados e estatísticas
- **Artilheiros** — maiores goleadores da história do torneio
- **Design responsivo** — adaptado para desktop, tablet e mobile

## Como rodar localmente

```bash
# Clone o repositório
git clone https://github.com/JaazielDev/CopaDoMundo.git

# Acesse a pasta
cd CopaDoMundo

# Abra no navegador
open index.html
```

> Não é necessário instalar dependências — o projeto roda diretamente no navegador.
